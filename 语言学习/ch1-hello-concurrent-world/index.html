Ch1: Hello, Concurrent World!

Date: June 4, 2020  14:39:51 Category: 语言学习

In C++, using multiple threads isn’t complicated in and of itself;
the complexity lies in designing the code so that it behaves as intended.

什么时候使用并发？

  • for separation of concerns
    • GUI
  • for performance
    • server-end

什么时候不使用并发？

  • 代码复杂性 > 性能提升
  • 出bug几率 > 性能提升
  • OS创建thread的开销（栈空间、内核资源） > 上限
  • context switch的开销 > 性能提升

极简Hello world版本

  • <thread>中定义线程管理相关类
  • 每个线程都要有 initial function
    • 主线程的 initial function 是 main()
    • 其他线程的 initial function 在 std::thread 对象构造函数中声明
