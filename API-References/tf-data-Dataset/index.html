调包侠工具箱：tf.data.Dataset

Posted on 2020-06-29 Edited on 2021-03-24 In API References

这是个啥？

  • tf.data.Dataset包含数据集导入、预处理、导出的高级api
  • 适用于数据量可以分批导入内存的场景
  • 可以通过简单的api实现流水线处理
  • 这货本质是个封装程度非常高的IO Adapter

Dataset创建

与导入操作有关的api

  • 从python list导入
  • 从txt文件导入
  • 从TFRecord导入
  • 从多文件导入（正则表达式）
  • 其他导入方式
    • tf.data.FixedLengthRecordDataset
    • tf.data.Dataset.from_generator

一些优雅的导入姿势

  • 导入一个tuple
  • 导入两个tuple

预处理

Dataset级别的api

  • .apply()

    • Args:

      • transformation func, from Dataset to Dataset

    • Rets:

      • a new Dataset
  • .batch()
    • Args:
      • batch_size
      • drop_remainder=False
    • 新Dataset输出的数据新增一个维度，大小为batch_size
  • .shard()
    • Args:
      • num_shards
      • index
    • Rets:
      • 一个新的Dataset，只包含原Dataset的1/num_shards数据，而且数据在原Dataset中的下标% num_shards = index
  • .concatenate()
    • Args:
      • another Dataset
    • Rets:
      • a new Dataset
  • Dataset.zip()
    • Args:
      • (Dataset, Dataset, …)
    • Rets:
      • a new Dataset

字段级别的api

  • .filter()
    • Args:
      • predicate(func)
    • Rets:
      • a new Dataset, element by predicate is True
  • .map()
    • Args:
      • map_func
      • num_parallel_calls(=tf.data.experimental.AUTOTUNE)
      • deterministic
    • Rets:
      • a new Dataset
  • interleave()
    • Args:
      • map_func
      • cycle_length=AUTOTUNE
      • block_length=1
      • num_parallel_calls(=tf.data.experimental.AUTOTUNE)
      • deterministic=None
    • Rets:
      • a new Dataset

如何理解：

  1. 遍历原Dataset，用map_func函数处理每个字段，该过程并行度由cycle_length控制
  1. 遍历cycles，取每个cycle的block_length个元素后，换下一个cycle继续读取，环形遍历直到所有元素都被取出
  1. 最终获得flatten后的新Dataset

Dataset导出

  • .prefetch()
    • Args:
      • buffer_size
    • Rets:
      • a new Dataset

官方建议在所有Dataset的处理结束后加上.prefetch()

  • .shuffle()
    • Args:
      • buffer_size
      • seed=None
      • reshuffle_each_iteration=None
    • Rets:
      • a new Dataset

官方建议buffer_size >= Dataset数据量

  • .enumerate()
    • Args:
      • start=0
    • Rets:
      • a new Dataset，每个输出结果多了一个表示索引的维度
  • .as_numpy_iterator()
    • Rets:
      • a new Dataset
